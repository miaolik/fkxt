# 反馈系统 (fkxt)

ElainaBot_v2 插件：收集用户反馈、建议、Bug、修改请求和创意，支持查询处理进度、管理员回复，并提供 Web 管理面板。

## 安装

将 `反馈系统` 目录放入 ElainaBot_v2 的 `plugins` 目录即可。

## 用户指令

| 指令 | 说明 |
| --- | --- |
| `反馈 <内容>` | 提交一般反馈（发送「反馈」查看 MD 填写指引） |
| `建议 <内容>` | 提交功能建议 |
| `bug <内容>` | 提交问题/故障 |
| `修改 <内容>` | 提交修改请求 |
| `创意 <内容>` | 提交新点子 |
| `我的反馈` | 查看自己提交的反馈及处理状态 |
| `查询反馈 <编号>` | 查看单条详情和管理员回复 |
| `删除反馈 <编号>` | 删除自己的反馈 |

## 管理指令

| 指令 | 权限 | 说明 |
| --- | --- | --- |
| `回复反馈 <编号> <内容>` | 唯一管理员/主人 | 回复反馈 |
| `处理反馈 <编号> <状态>` | 唯一管理员/主人 | 状态: 待处理/处理中/已完成/已拒绝 |
| `删除反馈 <编号>` | 唯一管理员/主人 | 可删任意反馈 |
| `设置反馈管理员 <ID>` | 主人 | 设置唯一管理员 |

## 防刷机制

- 提交冷却（默认 60 秒）
- 每人每日上限（默认 5 条，0 = 不限）
- 内容长度限制（5-500 字）
- 内容过百度文本审核（默认开启，不合规拒绝提交）

## Web 面板

后台「反馈系统」页面：

- 统计卡片：全部反馈 / 待处理 / 已回复
- 反馈列表：状态/分类筛选、关键词搜索、分页
- 每条反馈可 回复 / 修改（内容、分类、状态）/ 删除
- 设置：总开关、审核开关、冷却、每日上限、唯一管理员、百度审核凭据
